PhP

  PHP is easy to learn server scripting language, and a powerful tool for making dynamic and interactive Web pages. It is a widely used and efficient alternative to competitors such as Microsoft's ASP. The stable release of php is PHP7. Although, The PHP development team announces the immediate availability of PHP 8.0.0. This release marks the latest major release of the PHP language. PHP code is implemented on the server and the result is returned to the browser as plain HTML. The full ellipsis of PHP is "PHP: Hypertext Preprocessor". It can be download for free from PHP website. It can contain text, HTML, CSS, JavaScript, and PHP code.

  PHP can generate dynamic page content and have extension ".php" the end of the php files. It can create, open, read, write, delete, and close files on the server. In addition, it can send and receive cookies, so it helps to load website faster.

  PHP can add, delete, modify data in database and it can encrypt data PHP runs on various platforms such as Windows, Linux, Unix, Mac OS X.
