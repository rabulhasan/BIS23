Node JS

  Node.js is designed to build scalable network applications. It was written by Ryan Dahl, an American software engineer, in 2009. It is a cross-platform, open-source, back-end, JavaScript runtime environment that executes JavaScript code outside a web browser. It has an event-driven architecture capable of asynchronous I/O. It can be implemented in Linux, macOS, Microsoft windows. The current version of node.js is node.js 15 and 16. After every six months, the authority activates LTS version and become ready for general use. Many gigantic corporations such as LinkedIn, Microsoft, Netflix, PayPal, IBM use Node.js software. It could be misleading that the name “Node.js” does not refer to a particular JavaScript file though .js is a filename extension for JavaScript. It was designed with streaming and low latency keep in mind, so, it makes Node.js well suited for the foundation of a web library or framework. Node.js has a strong community base. A contributor, a collaborator, an observer and a member are the four types of involvement in the community. They altogether initiate educate the community, cultural direction of node.js foundation. In addition, the involve in translation and internationalization. Anyone can download node.js software from here. The Node.js distributed development project was previously governed by the Node.js Foundation, and has now merged with the JS Foundation to form the OpenJS Foundation.
